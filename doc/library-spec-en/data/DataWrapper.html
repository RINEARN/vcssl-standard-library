DataWrapper

Abstract

The data.DataWrapper library provides functionality for wrapping values of arbitrary types in a struct.It is mainly used for wrapping array data.

In VCSSL, jagged arrays are not allowed,so the number of elements in an array must be uniform within each dimension.
For example, multiple one-dimensional arrays with different numbers of elementscannot be combined into a two-dimensional array.

By wrapping array data in a struct,this restriction can be avoided.
In the above example, by creating a struct that wraps each one-dimensional arrayand then creating an array of those structs,a structure like a two-dimensional jagged array can be realized.

For such wrapping purposes, this module providesthe generic struct DataWrapper.
The wrap function wraps data of an arbitrary typeand converts it to a DataWrapper struct.The unwrap function can also be used to extract the original data.

DataWrapper is actually used in the implementations ofdata.WrappedList,WrappedStack, andWrappedQueue.
In those libraries, DataWrapper is used to realize collection functionalitythat supports arrays having different numbers of elements.
