List

Abstract

The data.List library provides a list-type data structure.

This module provides the generic struct List,which stores list data,and generic functions such as add and removefor adding and removing data from it.

Note that arrays cannot be handled as elements in data.List.If you want to use arrays as elements, usedata.WrappedList.
