Queue

Abstract

The data.Queue library provides a queue-type data structure.

This module provides the generic struct Queue,which stores queue data,and generic functions such as enqueue and dequeuefor adding and removing data from it.

Note that arrays cannot be handled as elements in data.Queue.If you want to use arrays as elements, usedata.WrappedQueue.
