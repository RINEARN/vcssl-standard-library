Stack

Abstract

The data.Stack library provides a stack-type data structure.

This module provides the generic struct Stack,which stores stack data,and generic functions such as push and popfor adding and removing data from it.

Note that arrays cannot be handled as elements in data.Stack.If you want to use arrays as elements, usedata.WrappedStack.
