WrappedQueue

Abstract

The data.WrappedQueue library provides a queue-type data structure.

This module provides the generic struct WrappedQueue,which stores queue data,and generic functions such as enqueue and dequeuefor adding and removing data from it.

In data.WrappedQueue, elements are implemented as values wrapped and storedin the DataWrapper struct.
Therefore, unlike data.Queue,WrappedQueue can handle arrays as elements.

On the other hand, because adding and removing elements involves wrapping theminto the DataWrapper struct, processing becomes heavier than in Queue by that overhead.
Therefore, if the elements are not arrays,using Queue is recommended rather than WrappedQueue.
