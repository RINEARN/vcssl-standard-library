WrappedStack

Abstract

The data.WrappedStack library provides a stack-type data structure.

This module provides the generic struct WrappedStack,which stores stack data,and generic functions such as push and popfor adding and removing data from it.

In data.WrappedStack, elements are implemented as values wrapped and storedin the DataWrapper struct.
Therefore, unlike data.Stack,WrappedStack can handle arrays as elements.

On the other hand, because adding and removing elements involves wrapping theminto the DataWrapper struct, processing becomes heavier than in Stack by that overhead.
Therefore, if the elements are not arrays,using Stack is recommended rather than WrappedStack.
