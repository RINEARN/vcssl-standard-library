HorizontalSliderWindow

Abstract

This library, gui.HorizontalSliderWindow, provides a window containing arranged horizontal sliders as a GUI component.

In the standard VCSSL library GUI, GUI components are handled as int IDs, but the GUI component provided by this module is handled as theHorizontalSliderWindow struct.

The HorizontalSliderWindow struct internally stores resource IDs and related data, but you normally do not need to be aware of its internals becausefunctions for creating and destroying it are provided. Direct access to its members is not recommended.
Specifically, create it with the newHorizontalSliderWindow function and destroy it with thedeleteComponent function.

This module also provides overloads of standard GUI library functions so that the HorizontalSliderWindow struct can be handled almost in the sameway as in the standard library GUI.
For example, the mountComponent, setComponentSize, andsetComponentLocation functions are provided.
