ElementaryFunction

Abstract

This library, math.ElementaryFunction, provides elementary mathematical functions.
Its role is similar to that of the standard VCSSL library Math,and it is intended to supplement functions not provided by Math.

This module also provides arbitrary-precision functions such as those for the varfloat and varcomplex types.
However, only the algorithms themselves are implemented, and no special measures are taken against the accumulation of calculation errors or loss of significance.

In other words, the returned value is not guaranteed to be correct to the last digit,and the tail of the result always contains an incorrect part caused by accumulated errors.
The length of this erroneous part varies depending on the algorithm used,the convergence speed for the given arguments, the degree of loss of significance, and other conditions.
For example, convergence generally becomes significantly slower near singular points, and accumulated errors also increase accordingly.

Therefore, when using arbitrary-precision functions such as those for the varfloat and varcomplex types,
call the function with a sufficiently larger number of digits than actually required,and discard the unnecessary part containing errors from the obtained result.
