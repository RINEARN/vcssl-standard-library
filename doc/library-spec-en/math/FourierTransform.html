FourierTransform

Abstract

This library, math.FourierTransform, provides processing for discrete Fourier transforms, or DFTs.

At present, the fast Fourier transform, FFT, is not implemented,and only a simple DFT with computational order N^2 is implemented.

This module also provides arbitrary-precision discrete Fourier transform processing with the varcomplex type.
However, only the algorithms themselves are implemented, and no special measures are taken against the accumulation of calculation errors or loss of significance.

In other words, the returned value is not guaranteed to be correct to the last digit,and the tail of the result always contains an incorrect part caused by accumulated errors.
The length of this erroneous part varies depending on the algorithm used and on differences such as loss of significance and other effects caused by the arguments.

Therefore, when using the arbitrary-precision DFT with the varcomplex type,
call the function with a sufficiently larger number of digits than actually required,and discard the unnecessary part containing errors from the obtained result.

Description Performs a discrete Fourier transform.The definition domain is from 0 to 2pi, and no normalization is performed after the transformation.
Parameters (complex[] type) arg : The data to be transformed by the discrete Fourier transform.
(int type) sgn : The exponent sign of the Fourier transform. Use -1 for the forward transform and 1 for the inverse transform. Normally specify the constant FourierTransform.FORWARD for the forward transform or FourierTransform.BACKWARD for the inverse transform.
Return (complex[] type) The value obtained by applying the discrete Fourier transform to the argument.
