Graph2D

Abstract

The tool.Graph2D API library, hereafter the Graph2D library, is a library for controlling 2D graph software.

The Graph2D library is an extension library and is not included in the standard libraries defined by the VCSSL language specification. Therefore, support for the Graph2D library implementation is not guaranteed in every VCSSL implementation.

The specification of the Graph2D library is defined assuming control of widely used general-purpose 2D graph software from VCSSL, and it is not intended for a specific 2D graph application. In an implementation of the Graph2D library, what kind of 2D graph software is adopted as the control target depends on the implementation, the vendor of the implementation, its version, or plugins that provide communication functions with graph software. Because of these circumstances, the functions provided by the Graph2D library should not be expected to behave in exactly the same way across different implementations.

Even in the same implementation, because the Graph2D library is not a standard library, its implementation is not guaranteed to behave permanently in the same way. Keep in mind that the implementation of the Graph2D library is only an extension feature supported independently by the implementation. For example, if the graph software receives a major update, the behavior of the Graph2D library may also change slightly as a result.

In the RINEARN VCSSL Runtime and VCSSL-supported software equipped with it, such as Rinearn Processor,the Graph2D library is supported, and its implementation adopts the graph software RINRARN Graph 2D for scientific and technical fields.Since the RINRARN Graph 2D program itself, RinearnGraph2D.jar, is bundled in the distribution package of the VCSSL Runtime,nothing needs to be prepared separately.
