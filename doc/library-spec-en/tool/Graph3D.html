Graph3D

Abstract

The tool.Graph3D API library, hereafter the Graph3D library, is a library for controlling 3D graph software.

The Graph3D library is an extension library and is not included in the standard libraries defined by the VCSSL language specification. Therefore, support for the Graph3D library implementation is not guaranteed in every VCSSL implementation.

The specification of the Graph3D library is defined assuming control of widely used general-purpose 3D graph software from VCSSL, and it is not intended for a specific 3D graph application. In an implementation of the Graph3D library, what kind of 3D graph software is adopted as the control target depends on the implementation, the vendor of the implementation, its version, or plugins that provide communication functions with graph software. Because of these circumstances, the functions provided by the Graph3D library should not be expected to behave in exactly the same way across different implementations.

Even in the same implementation, because the Graph3D library is not a standard library, its implementation is not guaranteed to behave permanently in the same way. Keep in mind that the implementation of the Graph3D library is only an extension feature supported independently by the implementation. For example, if the graph software receives a major update, the behavior of the Graph3D library may also change slightly as a result.

In the RINEARN VCSSL Runtime and VCSSL-supported software equipped with it, such as Rinearn Processor,the Graph3D library is supported, and its implementation adopts the graph software RINRARN Graph 3D for scientific and technical fields.Since the RINRARN Graph 3D program itself, RinearnGraph3D.jar, is bundled in the distribution package of the VCSSL Runtime,nothing needs to be prepared separately.

void setGraph3DCameraAngle( int graphID, float horizontalAngle, float verticalAngle, string mode )
Sets the camera angle.Specify the vector from the origin of the graph to the camera position, the camera-direction vector, by using the horizontal angle and the vertical angle based on the zenith axis described below for the horizontalAngle and verticalAngle arguments.The vertical angle specifies the angle between the camera-direction vector and the zenith axis.The horizontal angle specifies the direction of the projection of the camera-direction vector onto the plane perpendicular to the zenith axis.Here, the zenith axis can be selected from the X, Y, or Z axis by specifying one of the following values in the mode argument: ZENITH_X : An angle-specification mode in which the X axis is treated as the zenith axis , ZENITH_Y : An angle-specification mode in which the Y axis is treated as the zenith axis , ZENITH_Z : An angle-specification mode in which the Z axis is treated as the zenith axis ,
